Getting started

Thank you for using Slamby. We hope you will love what we do. We would like to help your work and make it smoother, so please check this tutorial and get a quick overview of our products.

In this short tutorial, we are going to show you how to use our Slamby API, Slamby Insight desktop application, which SDKs we provide and where you can find the full product documentation.

Install API server

You can use your Slamby API as a SaaS, or you can download, and host is by yourself. When you want to host it by yourself, the first thing you need to manage is the installation. To install Slamby API, please check the product readme and the installation guide.

Product documentation: https://github.com/slamby/slamby-api

Using Docker Compose, here are the installation steps:

  1. Install Docker on your machine: [Official Docker installation guide](https://docs.docker.com/engine/installation)
  2. Install Docker Compose on your machine (minimum 1.9.0 required): [Official Docker Compose installation guide](https://docs.docker.com/compose/install)
  3. Download our Docker Compose file
  4. ...

API setup

Great, you have a working Slamby API. Now your first step is the setup process.
During the setup process you need to:

  • Request a Slamby License,
  • Copy your Slamby API License,
  • and set the secret (password) for your Slamby API.

License
For open-source projects, you can use an `open-source license` for free, or for commercial purpose you can use a commercial license.

For open-source license, your can request one from the setup wizard directly by typing your e-mail address and click the `Request a License` button. Our license service will generate an open-sourced license for you and will send it via email to the given email address.

When your product or company is not open-sourced you cannot use Slamby API with open-source license; you need to purchase a `commercial license`. The pricing is based on the processor core number.

For pricing details, please check our pricing [https://www.slamby.com](https://www.slamby.com)

For purchasing a commercial license contact our [sales@slamby.com](mailto:sales@slamby.com)

Support

Slamby support mailing list for basic support: [https://groups.google.com/forum/#!forum/slamby](https://groups.google.com/forum/#!forum/slamby)

Each Slamby license covers a `basic support` with absolutely no SLA. This means you can access our support via our online support mailing list and you can ask for help from our community. In this case, the reaction time is not guaranteed, and there is no direct access to our technical experts.

When you need a `professional support`, please purchase one. Each professional support contains short-time SLAs and response time. Each professional support gives you direct access to our technical experts.

For more details, please check our available support options here: [https://www.slamby.com](https://www.slamby.com)

And contact our sales for more information: [sales@slamby.com](mailto:sales@slamby.com)
